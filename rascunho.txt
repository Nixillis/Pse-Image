* Filtro de média com máscara 

O que é : O filtro de média é um filtro de linear de suavização que trabalha diretamente
nos valores de média dos pixels, geralmente conhecidos como filtros passa-baixa.

Para que serve : Usado para que ocorra a perda de nitidez da imagem em que esta sendo 
aplicado


Como funciona : Ao aplicar o filtro de média, o valor de cada pixel analisado,  
tendo assim o valor de cada um deles substituido pela média dos pixels,que são
vizinhos a estes.

* Filtro Sobel

O que é : O filtro Sobel é um filtro para detecção de bordas, realçando linhas horizontais
e verticais mais escuras que o fundo, sem realçar pontos isolados.

Para que serve :Serve para realçando linhas horizontais e verticais mais escuras 
que o fundo, sem realçar pontos isolados.Destaca as bordas das imagens em que sua
mascara é aplicada.

Como funciona : Para se utilizar o filtro Sobel é necessário o uso de duas matrizes
3x3 que são convolvidas com a imagem original,sendo 1 delas para as variações horizontais
e outra para as verticais.Isso são mascaras.Fazendo a convolução dessas máscaras com a imagem
gera-se os gradientes vertical e horizontal da imagem que depois são combinados 
como mostrado.Além de calcular o gradiente, essas máscaras tem também o
efeito de suavização. Na prática ainda é necessário realizar a suavização da
imagem para eliminar ruídos.

* Alterar imagem pra tons de cinza

O que é : Conversão de uma imagem para tons

Para que serve : Geralmente se aplica para calcular a intensidade da luz em cada
pixel em cada faixa do espectromagnético de uma imagem.

Como funciona : Para se converter uma imagem em tons de cinza devesse utilizar uma
escala RGB, adicionando 30% do vermelho,59 % do verde e 11 % do azul independente da 
escala.O resultado é o nivel desejado de cinza.

* Pegar o negatico da imagem

O que é ? É uma função linear inversa, em que o contraste em áreas escuras tornem-se 
claras, e vice-versa.

Para que serve ? Serve para preservar a imagem, além de poder recupera-la caso ocorra 
a perda da mesma.

Como funciona ? Aplicasse o contraste em áreas de baixos valores de cinzas se tornem
altos valores de cinza e vice-versa. Sua função pode ser representada por

Y = - (AX + B)
onde
Y = novo valor de nível de cinza
X = valor original de nível de cinza
A = inclinação da reta (tangente do ângulo)
B = fator de incremento, definido pelos limites mínimo e máximo fornecidos pelo usuário.

* Interpolação bilinear

O que é ? A interpolação bilinear é uma extensão da interpolação linear,sendo esta,
a interpolação de duas funções com duas variáveis.Basicamente é a interpolação linear
aplicada duas vezes, contudo em direções diferentes.

Para que serve ?É uma técnica  de revezamento,usada para se ter uma imagem um pouco
mais realista.A interpolação bilineira pode ser usada onde a transformação de imagem perfeita 
com a correspondência de pixels é impossível, de modo que se pode calcular e atribuir valores 
de intensidade apropriados aos pixels.

como funciona ? Basicamente ele irá mapear uma localização de pixel de tela para um ponto
correspondente no mapa de textura.Uma média ponderada dos atributos como cor,transparencia
,etc dos quatro texels adjascentes é calculada e aplicada ao pixel da tela.Este processo
é repetido para cada pixel que forma o objeto sendo texturado.



* Segmentação
O que é ? É o processo de partição de uma imagem digital em varios segmentos(conjunto de pixels)

Para que serve ??Seu objetivo é simplificar e/ou alterar a representação de uma imagem em algo
que seja mais significativo e mais facil de analisar.


Como funciona ?A segmentação localiza na imagem objetos e limites como linhas e curvas nas imagens
Ela rotula cada pixel em uma imagem, de modo que os pixels com o mesmo rótulo compartilhem certas
caracteristicas como cor,intensidade ou textura.O resultado disso é um conjunto de segmentos que coletivamente cobrem toda a imagem ou
um conjunto de contornos extraidos da imagem.
